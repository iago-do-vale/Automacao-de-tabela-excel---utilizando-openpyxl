Este projeto realiza a automação de processamento de dados em planilhas Excel utilizando Python e a biblioteca openpyxl.

O script lê uma planilha contendo registros de vendas e organiza automaticamente os dados em abas separadas,
criando uma aba específica para cada produto. Caso a aba ainda não exista, ela é criada dinamicamente com a estrutura adequada.

Durante a execução, o sistema percorre toda a base de dados, 
identifica os produtos e transfere as informações correspondentes (data, categoria, quantidade, preço e total) para suas respectivas abas, mantendo a organização e facilitando a análise dos dados.
